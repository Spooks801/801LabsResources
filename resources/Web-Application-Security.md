*Resources dedicated specifically to hacking websites, SQL injection, XSS, and web protocols.*

* **[PortSwigger Web Security](https://portswigger.net/web-security/dashboard)** - Created by the authors of Burp Suite and the "Web Application Hackers Handbook." By far the most in-depth web security resource available.
* **[Practical Web Pen Testing (YouTube)](https://youtube.com/playlist?list=PLJnLaWkc9xRitBGPK5CcyoWCcPCeTYoqw&si=vQQ-6Ji9HP1ilG32)** - A great free course that goes over the basics of web penetration testing.
* **[Taggart Institute (PWST)](https://taggartinstitute.org/p/pwst)** - Practical Web Security Training course.
* **[OWASP WebGoat](https://owasp.org/www-project-webgoat/)** - A deliberately insecure web application maintained by OWASP for educational use.
* **[Hacker101 CTF](https://ctf.hacker101.com/)** - Web-focused CTF by HackerOne; solving levels gives you private bug bounty invites.
* **[PentesterLab](https://pentesterlab.com/exercises)** - Structured exercises and labs for web penetration testing.
* **[Rubiya CTF](https://los.rubiya.kr/)** - (Lord of SQLInjection) A specialized platform dedicated to SQL Injection attacks.
